# [audio-editor](https://hai2007.github.io/audio-editor/)
在线H5音频编辑器

> 本项目关于声音处理的底层依赖基于[audio.js](https://github.com/hai2007/audio.js)开源项目！

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/hai2007/audio-editor/issues)，欢迎参与维护！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```
